# Project: Library

This is a complete JavaScript library application built as part of [The Odin Project](https://theodinproject.com) curriculum. The main goal of this project is to practice working with Object-Oriented Programming (OOP), specifically constructors, prototypes, and storing data locally.

## 📌 Project Overview
The application allows users to keep track of their personal book collection. Users can add new books, delete existing books, and toggle the read status (Read/Unread) directly from the user interface.

## ✨ Features
- **Add New Books:** Opens a form/modal allowing users to input the Title, Author, Number of Pages, and Read Status.
- **Delete Books:** Remove any book from the library list and user interface.
- **Toggle Read Status:** Easily update whether a book has been read or not with a simple click.
- **Persistent Data (Optional):** Utilizes browser `localStorage` to save the library data, ensuring your list isn't lost upon page refresh.

## 💻 Technologies Used
- **HTML5**
- **CSS3** (with Flexbox/Grid)
- **Vanilla JavaScript** (ES6+)

## 🚀 Getting Started
To view and run this project locally on your machine, follow these simple steps:

### Prerequisites
You will need a modern web browser (like Chrome, Firefox, or Edge) and a code editor (like VS Code).

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/hgb131012/Library_Project.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Library_Project
   ```
3. **Open the project:**
   Double-click the `index.html` file to open it directly in your web browser.
